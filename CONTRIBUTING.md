# Contributing

Thank you for your interest in contributing to this project.

This repository contains an experimental research implementation exploring inference-time behavioral steering in large language models using contrastive activation directions.

Contributions that improve clarity, reproducibility, or experimentation are welcome.

---

## Ways to Contribute

You can contribute by:

- reporting bugs or unexpected behavior
- suggesting improvements to the methodology
- improving documentation or explanations
- proposing additional evaluation metrics
- experimenting with new models or layers
- adding reproducible experiments

---

## Reporting Issues

If you encounter a bug or unexpected result, please open an issue and include:

- a clear description of the problem
- steps to reproduce the issue
- the model used
- relevant logs or error messages

---

## Pull Requests

If you would like to contribute code:

1. Fork the repository
2. Create a new branch
3. Make your changes
4. Submit a pull request with a clear description

Please keep pull requests focused and well documented.

---

## Research Context

This project is an **experimental research prototype** exploring inference-time control mechanisms for large language models.

Results may vary across models and configurations.

---

## Code Style

Please aim for:

- clear and readable code
- descriptive variable names
- concise documentation for experiments

---

## Questions

If you have questions about the research or implementation, feel free to open a discussion or issue.
